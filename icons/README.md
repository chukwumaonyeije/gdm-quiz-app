# PWA Icons

This directory contains the Progressive Web App icons for the GDM Quiz App.

## Required Icons

You need to create the following icons:
- `icon-192x192.png` - 192x192 pixels
- `icon-512x512.png` - 512x512 pixels

## Design Guidelines

**Icon Theme:** Medical/Healthcare
- Use clinical blue/teal colors (#0ea5e9)
- Include elements suggesting:
  - Pregnancy/maternal health
  - Blood glucose monitoring
  - Medical education
  
**Recommended Elements:**
- Heart with pulse line
- Glucose meter icon
- Medical cross
- Quiz/question mark symbol

## Generation Options

1. **Online Tools:**
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator

2. **Design Software:**
   - Figma, Adobe Illustrator, Canva
   - Export as PNG with transparent background

3. **AI Generation:**
   - Use DALL-E, Midjourney, or similar
   - Prompt: "Medical app icon for gestational diabetes quiz, clinical blue color, simple flat design, 512x512"
